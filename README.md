# OLOD Digital Accessibility Brochure

A single-page, accessible HTML brochure about the Office of Learning and Organizational Development (OLOD) and its 2026 digital accessibility internship project at the University of Georgia.

## What this is

This page introduces OLOD's work with UGA Cooperative Extension, and shares the internship project on digital accessibility: what it set out to do, the work completed, and why it matters under ADA Title II.

## Accessibility

This page was built to follow WCAG accessibility standards, including:

- Real heading structure (H1, H2, H3) for easy navigation with screen readers
- A "skip to main content" link for keyboard users
- Text color contrast that meets WCAG AA/AAA levels
- Descriptive alt text on image
- No text placed only inside images
- Clear, simple language throughout
- Responsive layout that works on phones, tablets, and desktop
- Visible focus outlines for keyboard navigation
